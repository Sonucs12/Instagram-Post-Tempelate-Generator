## 🚀 Features
Here's what I've packed into this generator:

-   **Customizable Profile Section**: Upload your profile picture, enter a username, and add a verified badge if you want.
-   **Post Image Upload**: Easily add the image you want to use in your post.
-   **Interactive Customization**:
-   Change the number of likes displayed.
-   Set the location for your post.
-   Adjust the border color and width in real-time to match your style.
-   Choose from different image sizes (small, medium, or auto) to fit your needs.
-   **Preview Mode**: See exactly how your post will look before you download it.
-   **Downloadable Output**: Save your customized post as a high-quality image file.



## 🛠️ Technologies Used
I used these technologies to build this application:

-   **HTML**: The foundation for the application's structure.
-   **CSS**: For making the application look good and work on different screen sizes.
-   **JavaScript**: To make the application interactive and handle the dynamic updates.
-   **html2canvas**: This library is used to turn the HTML template into an image for downloading.

## 📘 Usage
I designed this tool to be super easy to use. Here's how to create your own Instagram post templates:

1.  **Open the Application**

Just open `index.html` in your favorite web browser.

2.  **Customize the Post**

Use the form on the right side to customize your post:

-   **Username**: Type in the username you want to display.
-   **Profile Picture**: Upload a profile picture that represents you or your brand.
-   **Verified Badge**: Check the box to add the verified badge, or uncheck to remove it.
-   **Post Image**: Upload the main image for your post.
-   **Location**: Enter the location you want to display.
-   **Likes Count**: Set the number of likes (up to 10,000) to show on the post.
-   **Border Customization**:
-   Pick a border color using the color picker.
-   Set the border width from 0 to 10 pixels.
-   **Image Size**: Choose the size of the final image (Small, Medium, or Auto).

3.  **Preview the Post**

Click the **Show Preview** button to see how your post looks. Keep tweaking the settings until you're happy with it.

4.  **Download the Post**

Click the **Download Post** button to save your customized post as an image file.

5.  **Use the Image**

Now you can share your downloaded post on Instagram or any other social media platform!
